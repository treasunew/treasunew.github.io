🚫注意事项

📍因为群里好像每天都有人有问题，群主解答也比较麻烦，所以统一归类汇总所有已解决问题

  1. 报错 如果里面有443 关闭防火墙，同时参考注意事项第十八条
  2. 建议定时
  4. token含义
  5. 重要提醒：第一个CK需要自己更改头部
  6. 脚本怎么放到青龙，打开脚本→复制→打开青龙→脚本管理→右边三个点（...）→添加→输入ksjsb.py→然后将复制的内容粘贴到里面，保存→定时任务→添加任务→名称随便填→命令：task ksjsb.py→定时请看第二条
  7. 以下操作导致黑号或者CK失效：
    • 退出登录
    • 修改头像，昵称等
    • 邀请新人
  8. 快手屏蔽了 腾讯云 阿里云的IP ,百度云虽然未屏蔽 但是极易被风控 所以要本地跑
  9. 每个月 提现100以上 要实名(实名最多2个号) 实名多了就黑 超过100 换快币
  10. 259金币为1 更换did或注销,1047金币为0,更换did或者停一天
  11. 变量ksjsbtask='yes' 就是强制开启所有任务,换did测试时候可以弄下,其他时间不要开启,自动优化任务(做完次数今日将不做)、优化网络连接,优化输出,增加token次数之类
  12. 出现如图情况，就是CK格式不对或者群主服务器网络波动，请检查CK（现在不能直接拼了，需要将对应参数替换进链接里面），或者等会儿再跑
  13. 若出现跑着跑着1047金币收益变为0得情况，一共有两种可能
    • 账号收益已经跑满
    • 若一开始跑就出现这种情况，请查阅变量表格，将强制任务开着跑，若还是没收益，那只能停一天再跑了，若停一天跑还是不行，那就是黑号了
  14. 若出现token出现错误
    • 请检查自己的变量名是否与变量表格内一致
    • 若一致，请检查token填写是否错误
    • 最好将变量放在配置文件中，用export ksjsbtoken = "你的token"
  15. 出现如图的情况，请到青龙的依赖管理→添加依赖→python3→输入requests→确定，即可跑脚本
  16. 若出现 no such file or directory 请参考注意事项第六条，然后新建任务即可
  17. 经过测试，did和rdid可以填写为一个，脚本正常使用
  18. 出现如图情况就是IP被快手ban了
    • 更换IP段
    • 疯狂高并发（开玩笑）
    • 重新跑（貌似没用）
  19. 账号黑了，自己注意提现的ip和对应的手机归属地要一样
  20. 返回 都是64,为账号异常 得实名

💡抓包相关

🎯一、抓包准备篇

  1. 安卓手机：安卓11及其以上需要root，安卓10以下可以直接安装证书到根目录，安卓对版本没要求（默认最新版即可）
    • 无Root获取CK
  2. 苹果手机：需要降级App才能抓到salt，且需要搭配安卓did才可以跑脚本
    • 工具：KNOT STREAM 在AppStore搜索安装即可
    • 版本：9.1.11.626
    • 抓登录的包
    • 降级教程 https://www.52pojie.cn/thread-1284776-1-1.html?tdsourcetag=s_pctim_aiomsg
  3. 安卓需要额外准备小黄鸟 https://wfwl66.lanzoul.com/i3mi1ux92aj 这里提供给大家下载
  4. 考虑到大家不止一个号，所以给大家提供一个分身工具 小X分身

注意：任何与root相关的问题请自己百度或者去米安找答案
好了，准备工作差不多就这样，我们直接进入正题

📱二、安卓篇

注意：最好直接抓链接里面含有client_salt的整段链接，自己设备如果不行可购买did更换did
例如：抓包完成之后，搜索：/rest/nebula/bubble/config，就可以抓到含有salt的链接

  1. 首先打开快手极速版，登录你所需要抓包的账号（安卓不需要抓登录包）然后划掉快手极速版的后台
  2. 然后打开小黄鸟，安装证书还有把证书放到根目录这些东西我就不讲了，然后点击侧边的三个横杠，再依次按图进行点击




    这样完成以后，回到最开始的界面，点击箭头指向的按钮就开始抓包了
  3. 打开快手极速版开始抓包，不出意外就会出现一堆数据包，然后搜索client_salt

🍏三、苹果篇

因为苹果抓包软件的安装和使用比较简单，我就不讲安装和使用了
大致流程就是安装→安装证书→信任证书→允许添加VPN
步入正题：

  1. 抓登录的包
  2. 具体版本自己看介绍，如果不在低版本好像抓不到包，降级请看准备工具和环境
  3. 苹果抓到的salt名字和安卓不一样，请更正为安卓格式client_salt，同时请更换did为安卓did（如自己无法获取，请找网站自行购买）
