2.安卓刷机

准备工作

  1. 一部安卓手机，系统最好Android10以上，此处我用RedMiNote7Pro刷机的
  2. 一台Windows电脑
  • 下载好AndoridSDKtool，里面有我们需要用到的fastboot和adb工具
  • 下载好PE包，可以先解压好，待会儿会用到
  • 下载搞机助手，安装目录自己选择
  • 下载TWRP刷机工具
刷PE系统
  1. 首先就是要解锁我们的手机，去小米解锁下载解锁工具，然后解压： 双击打开登录，点击解锁即可
  2. 解锁了之后，先长按音量-和关机键进入BootLoader模式
  3. 连接电脑，打开搞机助手，点击引导模式-刷入REC-选择Recovery，选择我们准备工作里面下载的twrp-xxx.img，然后刷入就行，刷入之后会Reboot进入Recovery模式
  4. 进入之后，selectLanguage为简体中文-然后将滑块滑到最右边，点击清除-点击格式化Data-再到挂载-停用MTP-再取消勾选Data-再到清除，点击格式化Data，按照提示操作；回到主页，点击重启，点击Recovery
  5. 按照以上的操作再次进入Recovery，点击安装，回到电脑上将下载的Pixel包复制到手机上，再回到手机上选择pixel-experience包，如果没有就回到主页再重新进安装就出来了，安装即可
  6. 跑完了就进入了pixel系统，按照提示可以设置语言然后选择在离线状态下设置，按照提示设置即可
获取ROOT

  1. 查看Ramdisk是否为是/否
    • 如果您的设备有Ramdisk，获取pixel-experience包的boot.img（或者init_boot.img如果存在）。
    • 如果您的设备没有Ramdisk，请获取recovery.img.
    • 您应该能够从官方固件包或自定义 ROM zip 中提取所需的文件
# 如果之前修补 boot.img 文件请用这个命令
fastboot flash boot magisk_patched_[random_strings].img
# 如果之前修补 init_boot.img 文件请用这个命令
fastboot flash init_boot magisk_patched_[random_strings].img
# 如果之前修补 recovery.img 文件请用这个命令
fastboot flash recovery magisk_patched_[random_strings].img

  1. 选择自己设备应该用的img然后将其复制到手机里，打开Magisk应用，选择安装-安装到Recovery-选择并修补一个文件-选择刚才复制到手机的img-然后点击开始，等它打包完成，到它输出的目录，将其复制到电脑上

  2. 将手机进入Recovery模式-高级-Sideload（这时不会有任何进度条，它是静止的），打开搞机助手-选择恢复模式-Sideload刷机模式-将经过Magisk打包的img选中-开始刷机。不出意外，这里就会成功获取ROOT权限了

到此就结束了，请尽情地玩耍吧

3.Git拉取仓库

我的环境：

Windows10 ClashVerge

首先说明我遇到的问题：我在有科学上网的环境下依旧没法克隆、推送Repo到Github 于是我在网上查了下资料参考一和参考二有如下解决办法

  1. 适用于本地有科学上网环境
#设置代理
git config --global http.proxy http://127.0.0.1:7890
git config --global https.proxy http://127.0.0.1:7890
git config --global http.proxy 'socks5://127.0.0.1:7891'
git config --global https.proxy 'socks5://127.0.0.1:7891'
# 取消代理
git config --global --unset http.proxy
git config --global --unset https.proxy
  1. 适用于自己有架设代理服务器，如socks5和http
#设置代理
git config --global http.proxy http://proxyUsername:proxyPassword@proxy.server.com:port
git config --global https.proxy http://proxyUsername:proxyPassword@proxy.server.com:port
git config --global 协议.proxy 协议://ip地址:端口号
# 取消代理
git config --global --unset http.proxy
git config --global --unset https.proxy

到此所有的设置都结束了，愉快的玩耍吧
