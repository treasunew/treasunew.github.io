  2. ROOT但无面具（Magisk）
    • 2.1 面具不是只你装个APP就行了，还得有Magisk环境，自己去米安找安装Magisk的教程
    • 2.2 装好面具之后参考前面的即可
  3. 无ROOT
    • 3.1 无ROOT有几种解决办法，第一种ROOT，第二种安装恢复包，第一种自己折腾，重点讲第二种
      • 3.1.1 如果安装出现下图所示报错，请安装旧版本termux(ZeroTermux_0.117.13.apk)，如果依旧报错，说明你的手机安卓版本过低，请自行前往官方下载站尝试安装旧版本
      • 3.1.2 如果问题依旧如此，说明你的手机安卓版本过低，无法安装termux，请更换高版本安卓系统！(也就是没救了)
      • 3.1.3 正式开始
      • uname -m 返回的不是aarch64，说明你的手机不是arm64的cpu，可以放弃安装

        链接：https://mdl.ink/S439cD
        密码：termux
      • 3.1.4小结
        导入青龙恢复包：
        侧滑->备份/恢复->恢复->选择一个青龙恢复包->给容器添加名称->恢复->等待恢复完成即可
        切换到青龙面板容器：
        侧滑->容器切换->选择刚刚恢复的容器名称->切换->需要


        手机打开浏览器访问127.0.0.1:5700
        熟悉的界面出现了，接下来该干啥就不用我多说了吧！
        青龙面板的所有文件均在alpine系统的/ql路径下，执行cd /ql 即可进入到青龙目录中

        四、使用FTP文件传输工具上传本地脚本到青龙内部文件夹中，废除软链接方法【请使用v3版恢复包】
        1、安装FTP文件传输工具【FileZilla_1.7.apk】，试了很多FTP工具，目前暂时只发现这个app能连上。并按照如下图方式设置好，最后点击保存
        2、快捷下载地址 https://grbnb.lanzouf.com/i27MQ04m0fcj

        3、点击连接即可连接进入到termux内部文件夹中

        4、进入到alpine系统后再点击进入到ql文件夹，此处是整个青龙面板的工作目录

        5、把本地脚本文件上传到scripts或者raw任意一个文件夹中，但是只有scripts文件夹中的脚本可在web浏览器面板的脚本管理中查看。你想上传脚本到哪个文件夹就先进入到这个文件夹中，然后再点击上传，再在本机内部存储中【默认是Download目录】找到你要上传的脚本所在的路径点击右上角的确定即可！
        6、回到青龙面板，在定时任务中新建任务，参考下图填写本地脚本定时运行任务
  • 青龙v2.10.13版本的脚本文件在当前工作目录下的scripts、raw文件夹中。v2.12.0版本以上均在data目录中

💻电脑DockerDesktop

  1. 下载安装之后，打开软件
  2. 因为Docker自带的镜像源太慢，所以更换下镜像源（有科学上网环境不用换），参考https://blog.csdn.net/weixin_39305029/article/details/105014150
  3. 然后使用 Win+R 输入cmd打开终端
  4. 构建青龙容器
    • cd 你想把青龙本地目录放哪儿就写那个路径
    • docker pull whyour/qinglong
    • docker run -v $PWD/qinglong:/ql/data -p 5700:5700 --name qinglong --hostname qinglong --restart unless-stopped whyour/qinglong:latest
    • 这样，本地青龙就搭建完成了，自己愉快的玩耍吧
